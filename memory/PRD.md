# UGG - Universal Gaming Gateway | PRD

## Architecture
- **Backend**: FastAPI + MongoDB — 32 route modules, 240+ API endpoints
- **Gateway Core**: 8-stage pipeline | **Adapters**: SAS/G2S/S2S/VCF
- **Frontend**: React 19 + Tailwind + Recharts + Mapbox GL + Framer Motion — 28 pages
- **AI**: Gemini 3 Flash (analytics + connector gen + chat) | **Auth**: JWT RBAC (7+4 tier) | **Real-time**: WebSocket
